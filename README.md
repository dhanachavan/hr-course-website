# Manufacturing Training Portal

A single-page website for the Head of Training at a manufacturing company, featuring an Apple-inspired design. This website allows users to explore and choose different types of training courses.

## Features

- Clean, modern Apple-esque design with plenty of white space and subtle gradients
- Responsive layout that works on all devices
- Course filtering functionality to browse different training categories
- Contact form with validation
- Newsletter subscription
- Smooth scrolling and animations

## Design Elements

The website incorporates Apple-inspired design principles:
- Minimalist aesthetic with ample white space
- Subtle gradients and shadows
- Smooth transitions and animations
- Elegant typography using SF Pro Display and Helvetica Neue
- High-quality images and icons

## Website Sections

1. **Header/Navigation**: Fixed navigation with smooth scrolling links
2. **Hero Section**: Prominent headline and call-to-action
3. **Courses Section**: Filterable grid of training courses
4. **About Section**: Information about the training program with key statistics
5. **Contact Section**: Contact form and company information
6. **Footer**: Additional links, newsletter signup, and social media

## Course Categories

The website showcases four main course categories:
- Technical
- Safety
- Leadership
- Quality Control

## Setup Instructions

1. Clone or download this repository
2. Add your own images to the `images` folder:
   - course-technical-1.jpg
   - course-technical-2.jpg
   - course-safety-1.jpg
   - course-safety-2.jpg
   - course-leadership-1.jpg
   - course-leadership-2.jpg
   - course-quality-1.jpg
   - course-quality-2.jpg
   - about-training.jpg
3. Open `index.html` in your web browser

## Customization

- **Colors**: Edit the CSS variables in the `:root` section of `css/styles.css`
- **Content**: Modify the text in `index.html`
- **Images**: Replace placeholder images in the `images` folder
- **Additional Courses**: Add more course cards in the `course-grid` section

## Technologies Used

- HTML5
- CSS3 (with variables and flexbox/grid)
- JavaScript (vanilla, no frameworks)
- Font Awesome icons
- Google Fonts (SF Pro Display)
